## Примеры кода

Вот пример реализации алгоритма сортировки пузырьком на Python:
```
def bubble_sort(arr):

n = len(arr)

for i in range(n):

for j in range(0, n-i-1):

if arr[j] > arr[j+1] :

arr[j], arr[j+1] = arr[j+1], arr[j]
```
Пример использования:
